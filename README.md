# 🌱 Sintaxe do Vínculo

> "Palavras são janelas (ou paredes)." — Marshall Rosenberg

**Sintaxe do Vínculo** é um ecossistema de ferramentas minimalistas para o cultivo de relações conscientes. Unindo estrutura e empatia, o projeto oferece interfaces para manutenção de diálogo, quebra-gelos cotidianos e um formulador de Comunicação Não Violenta (CNV). Um repositório para organizar o invisível e transformar a reatividade em conexão.

---

## 🛠️ O Sistema

O projeto é dividido em três módulos principais, cada um projetado para uma camada da convivência:

1.  **🍽️ Quebra-gelo (Presença):** Perguntas aleatórias para momentos de pausa (Almoço & Janta), combatendo o modo automático e estimulando a curiosidade mútua.
2.  **🤝 Protocolo de Cuidado (Manutenção):** Uma interface com timers de tempo mínimo/máximo para ajustes emocionais e vida prática, garantindo profundidade sem exaustão.
3.  **💬 Tradutor de Conexão (CNV):** Um formulador inteligente baseado nos 4 pilares da CNV (Observação, Sentimento, Necessidade e Pedido) com feedback em tempo real para evitar julgamentos.



## 🧠 Filosofia do Projeto

Como estudante de **Ciência de Dados** e entusiasta da escrita, acredito que as relações humanas possuem uma "sintaxe". Assim como um código precisa de clareza para rodar sem erros, os vínculos precisam de intenção para não quebrarem sob o peso da rotina. Este repositório é o meu "ancoradouro" para documentar e facilitar esses processos.

## 🚀 Tecnologias

- **Frontend:** HTML5, CSS3 (Custom Properties & Flexbox), Tailwind CSS.
- **Lógica:** JavaScript Vanilla (ES6+).
- **Persistência:** LocalStorage para preservação de rascunhos.
- **Saída:** Integração com jsPDF para exportação de acordos e reflexões.

## 📂 Como usar

1. Clone o repositório: `git clone https://github.com/seu-usuario/sintaxe-do-vinculo.git`
2. Abra o arquivo `index.html` em qualquer navegador.
3. Não requer backend ou instalação de dependências (estático e leve).
