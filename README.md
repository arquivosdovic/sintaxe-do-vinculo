# sintaxe-do-vinculo
Sintaxe do Vínculo é um ecossistema de ferramentas minimalistas para o cultivo de relações conscientes. Unindo estrutura de dados e empatia, o projeto oferece interfaces para manutenção de diálogo, quebra-gelos cotidianos e um formulador de CNV. Um repositório para organizar o invisível e transformar palavras em janelas.
